# Security Policy

## Reporting a vulnerability
Please report security issues privately to **david@linacre.site** or via a
[GitHub security advisory](https://github.com/LIN4CRE/dkma-monster/security/advisories/new).
Do **not** open a public issue for undisclosed vulnerabilities.

We aim to acknowledge reports within 72 hours.

## Scope
This project is static documentation plus local automation scripts (ADB CLI,
desktop GUI, root/Magisk helpers). There is no server or user data. Please report:
supply-chain risks, malicious script behaviour, or issues in the generated site.
The security posture here is **defensive only**.
